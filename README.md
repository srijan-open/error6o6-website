# Error6o6 Website

Official landing page for Error6o6 Technologies.

Error6o6 is an AI-native product lab building frontier products across storytelling, intelligence systems, creative tools, and future product ecosystems. This repo currently contains the single-page public landing page at `/`.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- lucide-react

## Local development

```bash
npm install
npm run dev
```

## Verification

```bash
npm run lint
npm run build
```
